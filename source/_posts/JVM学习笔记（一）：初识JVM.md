## 1.JVM的概念
**1)虚拟机简介：**
1. JVM：
Java Virtual Machine的简称。意为Java虚拟机。
2. 虚拟机：
指通过软件模拟的具有完整硬件功能的、运行在一个完全隔离的环境中的完整计算机系统。
3. 常见的虚拟机：
	- JVM
	- VMwave
	- Virtual Box

**2)JVM和其他两个虚拟机的区别：**
1. VMwave与VirtualBox是通过软件模拟物理CPU的指令集
物理系统中会有很多的寄存器
2. JVM则是通过软件模拟Java字节码的指令集
JVM中只是主要保留了PC寄存器，其他的寄存器都进行了裁剪
3. JVM是一台被定制过的现实当中不存在的计算机。
